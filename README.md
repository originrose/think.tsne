# think.tsne

Thin clojure wrapper on top of:
https://github.com/lejon/T-SNE-Java

## Usage

In your project, using lein:
```
[thinktopic/think.tsne "0.1.1"]
```

Build this repo:

```
git submodule init
git submodule pull
lein test
```

Example:

[Unit Test](test/think/tsne/core_test.clj)
